О проекте
Arkanoid-SFML — это двухмерная игра в жанре аркады, разработанная на языке C++ с использованием мультимедийной библиотеки SFML. Проект представляет собой классическую механику уничтожения кирпичей с помощью мяча и подвижной платформы, построенную на объектно-ориентированном подходе и кастомной физике коллизий.
Игрок управляет платформой, перемещая её по горизонтали в нижней части экрана, чтобы отбивать мяч и направлять его в поле кирпичей. Цель игры — разрушить все блоки на экране, удерживая мяч в пределах игрового поля и зарабатывая очки. 

Архитектура и структура репозитория
Архитектура распределена по специализированным заголовочным файлам, каждый из которых отвечает за конкретную изолированную систему или сущность
